Question3 Solution

Solution1

If the Base.render() is given 1 argument 

		Child.prototype.render = function(data) {
			this.preRender();
			var base = Object.getPrototypeOf(this);
			base.render(data);
		}

Solution2

If the Base.render() is given extra arguments 

		Child.prototype.render = function() {
			this.preRender();
			var base = Object.getPrototypeOf(this);
			base.render.apply(base, arguments);
		}
I'm not sure what "given extra arguments" means. So I assume it means Base.render take more arguments like "Base.prototype.render = function(data1, data2){...}"
